IT Landscape

Document your entire information system

Applications, interfaces, and infrastructure in one place. Visualize your architecture with interactive maps and track lifecycles from proposal to retirement.

Application Portfolio Management

Maintain a comprehensive inventory of your applications with per-environment instances. Track ownership, criticality, compliance status, and lifecycle stage. Manage version lineage as applications evolve.

  • Logical applications with per-environment instances (prod, pre-prod, QA, dev)
  • Version lineage tracking for application evolution
  • Lifecycle management: Proposed, Active, Deprecated, Retired
  • Ownership tracking: IT owners, business owners, support contacts
  • Compliance fields: data class, PII flag, data residency

Interface Documentation

Document integrations between applications with business and technical definitions. Support for direct connections and 3-leg middleware patterns (Extract/Transform/Load). Configure per-environment bindings with endpoints and authentication.

  • Business purpose, data category, and process linking
  • Direct or via-middleware route types
  • 3-leg support: Extract, Transform, Load legs
  • Per-environment, per-leg binding configuration
  • Functional definition: business objects, use cases, key identifiers

Infrastructure Registry

Track servers, locations, and connections across your infrastructure. Support for on-premise and cloud deployments. Document network connectivity with multi-server topologies and layered routing.

  • Server registry: VMs, databases, queues, containers, functions
  • Location management: datacenters, cloud regions, availability zones
  • Connection documentation: server-to-server and multi-server mesh
  • Cluster support with member grouping
  • Operating system tracking with support dates

Visual Architecture Maps

Two interactive D3-powered visualizations help you understand your architecture at a glance. Filter by environment, lifecycle, or specific root nodes. Export to SVG or PNG for documentation.

  • Interface Map: Applications as nodes, interfaces as edges
  • Connection Map: Servers/entities with network connectivity
  • Business view (hides middleware) and technical view
  • Depth filtering: limit graph to N hops from selected nodes
  • SVG and PNG export at 2x resolution
