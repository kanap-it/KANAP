What Plaid Can Do

Not a chatbot. A real assistant.

Search & Query Everything

Ask Plaid about your applications, servers, contracts, budget items, projects, tasks, or documents. Get instant answers with structured data, not vague summaries.

  • Cross-entity search across all modules
  • Structured queries with filters and sorting
  • Aggregations and statistics
  • Knowledge base full-text search

Take Action

Plaid doesn't just read — it writes. Create documentation, update task statuses, reassign work, add comments, and manage document metadata, all from a conversation.

  • Create and edit documents with markdown
  • Update task status and assignees
  • Add comments to tasks
  • Preview changes before applying

Natural Conversation

No query language to learn. Just ask in plain language. Plaid understands context, follows up, and adapts to your role and permissions.

  • Context-aware: knows your role and team
  • Multi-turn conversations with memory
  • Respects RBAC — sees only what you can see
  • Available in English, French, German, Spanish

Instant Summaries & Reports

Get a project status briefing, a budget overview, or a list of expiring contracts — synthesized from live data, not a static dashboard.

  • Project health summaries on demand
  • Budget variance analysis in plain language
  • Application dependency overviews
  • Web search for external context

Secure by Design

Plaid inherits KANAP's row-level security. Every query runs within your tenant's permissions. No data leaks, no shortcuts.

  • Row-level security enforced on every query
  • User permissions respected at all times
  • API keys with granular scoping
  • Mutation preview before applying changes

Bring Your Own Key

No vendor lock-in. Connect OpenAI, Anthropic, Ollama, or any OpenAI-compatible provider. Your keys, your models, your choice.

  • OpenAI, Anthropic, Ollama built-in
  • Custom OpenAI-compatible endpoints
  • Encrypted key storage
  • Admin-controlled provider settings

Open Integration

MCP: Use KANAP data in any AI tool.

KANAP exposes a full MCP (Model Context Protocol) server. Connect Claude Desktop, Cursor, Windsurf, or any MCP-compatible tool — and query your IT governance data without leaving your workflow.

Your enterprise AI tools gain instant access to applications, budgets, projects, infrastructure, contracts, and documentation — all through a secure, authenticated API.

Real Examples

What you can ask Plaid.

💬 "What's the status of Project Atlas?"

Plaid pulls the project record, its tasks, timeline, team, and gives you a concise status brief — including blockers and upcoming milestones.

💬 "List all applications hosted on AWS with no documented owner"

Structured query across your IT landscape. Plaid returns filtered, sorted results from live data — not a static report.

💬 "Draft a security policy for remote access"

Plaid creates a new document in your knowledge base, with proper metadata and markdown content. Review it, edit it, approve it — all in KANAP.

💬 "Which contracts expire in the next 90 days?"

Instant answer from your contract registry, with supplier names, amounts, and renewal dates. No clicking through dashboards.

💬 "Reassign all of Marie's open tasks to Thomas"

Plaid shows you a preview of affected tasks, lets you confirm, then applies the changes in one go.

💬 "Summarize our IT budget variance vs. last year"

Plaid aggregates budget data across years and departments, then explains the variance in plain language — ready to forward to your CFO.
