# Manuel utilisateur KANAP

Bienvenue dans le manuel utilisateur KANAP. Cette documentation couvre toutes les fonctionnalités disponibles pour les utilisateurs de la plateforme KANAP.

## Premiers pas

KANAP aide les équipes IT et finance à gérer leur paysage technologique, leurs budgets et leurs opérations. Utilisez le menu de navigation pour explorer les différents modules.

Si vous souhaitez être productif le plus rapidement possible, commencez par les guides Fast Track avant de plonger dans les pages de référence complètes.
