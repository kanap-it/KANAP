# KANAP Benutzerhandbuch

Willkommen beim KANAP Benutzerhandbuch. Diese Dokumentation behandelt alle Funktionen, die Benutzern der KANAP-Plattform zur Verfügung stehen.

## Erste Schritte

KANAP unterstützt IT- und Finanzteams bei der Verwaltung ihrer Technologielandschaft, Budgets und des operativen Betriebs. Nutzen Sie das Navigationsmenü, um die verschiedenen Module zu erkunden.

Wenn Sie den kürzesten Weg zur produktiven Nutzung suchen, beginnen Sie mit den Schnelleinstieg-Anleitungen, bevor Sie in die vollständigen Referenzseiten eintauchen.
