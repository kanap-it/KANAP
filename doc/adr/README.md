# Architecture Decision Records (ADRs)

This folder contains Architecture Decision Records. Create one ADR per decision using the template below. Number ADRs sequentially.

## Usage
1. Copy `0000-template.md` to the next number (e.g., `0001-decide-x.md`).
2. Fill all sections; keep concise and reference detailed docs if needed.
3. Link the ADR from relevant docs (e.g., `architecture.md`).
